Hierarquia dos Sistemas Operacionais

Descubra a história por trás do código: explorando a evolução dos sistemas operacionais.

"A tarefa básica que um computador realiza é a execução de programas. O programa a executar consiste em um conjunto de instruções armazenadas na memória." (STALLINGS, 2018, p. 5)

O que é um Sistema Operacional?

"É difícil dizer com absoluta precisão o que é o sistema operacional, além de ele ser o software que opera em modo núcleo [...] cuja função é fornecer aos programas do usuário um modelo do computador melhor, mais simples e mais limpo, assim como lidar com o gerenciamento de todos os recursos mencionados." (TANENBAUM, 2015, pp. 1, 3)

Unix/Linux

POSIX

"Para tornar possível escrever programas que pudessem ser executados em qualquer sistema UNIX, o IEEE desenvolveu um padrão para o UNIX, chamado POSIX (Portable Operating System Interface — interface portátil para sistemas operacionais), ao qual a maioria das versões do UNIX dá suporte hoje em dia." (TANENBAUM, 2015, p. 10).

"O padrão define várias funções, como manipulação de arquivos, gerenciamento de processos e chamada do sistema APIs. POSIX incentiva a compatibilidade e operações entre plataformas em Sistemas Linux." (PhoenixNAP Global IT Services, 2022).

Windows 95:
Revolução dos PCs

"O sistema operacional da Microsoft foi lançado em 24 de agosto, há exatos 25 anos, com funcionalidades que resistem até hoje em grande parte dos PCs." (Wakka, 2020).

Com sua interface gráfica amigável, o Windows 95 popularizou o uso do mouse e introduziu recursos icônicos como o Menu Iniciar e a barra de tarefas. Essas funcionalidades resistem até hoje e influenciaram gerações de sistemas operacionais subsequentes
